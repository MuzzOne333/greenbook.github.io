Ремарка для тиммейтов!
Перед тем как запустить сайт, вам нужно находиться в данной директории: ~/FinalProject_WebTechnologies/animal-geographic
Как войдете в эту директорию, в терминале вводите следующую команду: "npm run dev"
В случае ошибки, следует сделать вывод, что у вас не установлен Node Js. Когда установите, ошибки уже не должно быть.
